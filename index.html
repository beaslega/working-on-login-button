Welcome to the Grave Site

The Grave Site

As you develope The Grave Site you will learn the basics of coding a web site. the course will cover many different tags but no one can remember all of them. If you need to look up or reference a tag or just remember what one does go look around w3schools.com.

Web Development Essentials

In Web essentials we will learn how to code the basic structure of the page using HTML5 as well as CSS3

Web Databases

In Web Databases you will learn how to setup and structure a Database of tombstones using MySql

Serverside scripting

In server side scripting we will be teaching you about PHP. PHP allows you to make a web site dynamic, as well as interact with, retrieve, and post data to the Database

Client Side Scripting

Client Side Scripting will cover JavaScript & JQuery adding additional functionality to the website and completing the basic language used in Web Development
By the end of this class you will have completed most of the core classes in the program
